#How can I ensure that my code, program, or software is functional and secure?
Using unit testing is a powerful way to ensure that each new element of a program is fully functional. I also incorporated secure coding practices and input validation.


#How do I interpret user needs and incorporate them into a program?
I interpret user needs by first understanding the problem that the user is trying to solve. I then turn those needs into requirements and then look to implement those.


#How do I approach designing software?
My approach is to take the requirements given and then break the problem into smaller pieces. Then I am able to chip away one at a time. This increases speed, efficiency, and simplicity.
